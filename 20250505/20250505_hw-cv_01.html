陳小美

我是一位專注於後端開發的軟體工程師，擁有，擅長使用 Node.js、Python 及 Java 開發高效能且可擴展的後端系統。從大學主修資訊工程以來，我便對系統架構設計與資料處理充滿興趣，畢業後積極投入軟體開發領域，持續精進技術與解決實際業務問題的能力。

在過去的工作經歷中，我曾，包括電商平台、內部管理系統與 API 設計專案。我熟悉 RESTful API 的設計原則，並具備實作 OAuth 2.0、JWT 等身份驗證機制的經驗。此外，我對資料庫的設計與優化有深入的理解，熟練操作 MySQL、PostgreSQL 及 MongoDB，能針對不同場景選擇合適的資料儲存方式與索引策略，以提升整體系統效能。

我也，透過監控工具如 Prometheus 與 Grafana，定期檢視系統運作狀況，主動找出瓶頸並進行調整。例如，在某次專案中，透過重構原本同步的 API 邏輯為非同步處理，加上快取策略的導入，使系統回應速度提升了超過 60%，並成功支撐高峰期的用戶流量。

在團隊合作方面，我，習慣使用 Git 進行版本控制，並搭配 GitLab CI/CD 完成自動化部署流程。我也參與過敏捷開發流程，熟悉 Scrum 方法中的每日站立會議、迭代開發與回顧機制，能快速配合跨部門團隊協作，依據需求調整開發優先順序與資源配置。

我持續關注後端技術的新趨勢，對有一定程度的了解與實作經驗。近期我也開始學習 Go 語言與 Redis Cluster 的應用，希望在大型系統設計與分散式架構上有更進一步的成長。

我相信後端工程師的價值不僅在於程式碼的產出，更在於如何設計出穩定、安全且易於維護的系統。我。未來我希望能加入一個重視技術、鼓勵創新的團隊，持續學習、貢獻所長，並與團隊一同打造具影響力的產品與服務。
